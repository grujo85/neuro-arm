# 🤖 NEURO-ARM v1.0
### Interactive 3D Robotic Arm Simulation

A lightweight, web-based robotic arm simulator developed using **Flask** and **Three.js**. This project demonstrates real-time joint visualization and 3D rendering directly in the browser.

---

## 🚀 Key Features
* **High-Fidelity 3D Rendering:** Smooth visualization using WebGL and the Three.js library.
* **Real-time Interaction:** Low-latency joint control (Base, Shoulder, Elbow) via intuitive UI sliders.
* **Responsive UI:** Cyberpunk-inspired dashboard designed for both desktop and mobile accessibility.
* **Zero Installation:** Runs entirely through the browser with no specialized hardware required.

## 🛠️ Tech Stack
* **Backend:** Python 3.12+ / Flask (Production-ready with Gunicorn)
* **Frontend:** JavaScript (Three.js), HTML5, CSS3 (Modern Dark Mode)
* **Deployment:** Render / GitHub CI/CD
